Overview

Generic Cam is a toolpath generator for CNC processing.

1 Introduction

A toolpath generator is the missing link between the model and machine.
To machine a model generated by some CAD software, You need a some G-Code program telling the machine how to move to produce the object desired.
Generic CAM is an open source tool path generator for CNC machines.
The "Generic" in Generic CAM stands for being universal. i.e. every type of CNC machine can be targeted (Multi axis: X, Y, Z, A, B, C, U, V, W). Generic CAM starts from CAD data provided in the STL -, GTS - or DXF - file format. Based on the capabilities of the selected CNC-machine G-codes are generated.
This documentation consists of two parts. The first one descripts the workflow. The second chapter assists on the writing of machine description files.

2 CNC Basics

2.1 Feeds and Speeds

  • Tool speeds are in SFM. Surface Feet per Minute. Speed of the blade in a material.
  • Calculate RPM from SFM and the tool diameter.
  • Forward speed: Chipload for every tooth (flute) on the cutter. Width of the chips. Not to far below the optimal value.
  • Climb milling: Chip from wide to small. Better surface.
  • Convential milling: Chip from small to wide. No problems with imprecise bearings.
  • RPM as fast as possible. But spindle rpm vs power < spindle max power.
  • Machine rigidity.
  • Cooling and coolant.
  • Slotting vs pocketing vs peripheral milling
  • Shortest cutter possible, widest diameter possible.
  • Quill [A]  [A] Quill: (german) Hohlwelle, Pinole
  • 4-6 mil Chipload for hardwood.
